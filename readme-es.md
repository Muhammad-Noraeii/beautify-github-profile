# 📌 Primer paso: Configurar el repositorio de GitHub  
Crea un repositorio con tu nombre de usuario en GitHub 👇

<img src="name.png">

Plantilla: 👇  
```
https://github.com/username/username
```

Ejemplo: 👇  
```
https://github.com/rzashakeri/rzashakeri
```

### Después de crear el repositorio, **crea un archivo README.md** en el repositorio y listo ✅  
Ahora que hemos creado el repositorio, llegamos a la parte interesante: **diseñar nuestro README.md.**

> Guía en persa para el primer paso 👉 [Enlace 🔗](https://www.instagram.com/p/CQlxnAnHId0/)
